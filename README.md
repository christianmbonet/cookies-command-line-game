# Cookies repl-game

## Description

Who stole the cookies from the cookie jar?

## Set Up

### Clone Repo

```
git clone git@github.com:christianmbonet/cookies-repl-game.git
cd cookies-repl-game
```

### Install Yarn

```
yarn
yarn add readline-sync
```

## To Play Game

```
node index.js
```

Enjoy!
